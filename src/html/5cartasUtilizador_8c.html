int valida_flush ( MAO  m)

Ao receber o highlight do utilizador (neste caso um flush), confirma se este é válido para ser efetuada uma jogada. Nesta função, é retornado 0,1,2 ou 3 conforme o naipe do flush. Exemplo: 0 -> Ouros; 1 -> Copas; etc... Com isto, conseguimos mais tarde avaliar dois flushes.

mA mão de um jogador.
Returns
